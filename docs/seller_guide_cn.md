# 卖家使用指南

你好！如果你正在出售二手显卡或电脑，买家可能会要求你运行这个工具来验证硬件真实性。

**不用担心，这个工具非常安全，只需要 10-20 分钟。**

---

## 这个工具是做什么的？

这是一个**验机工具**，帮助买家确认：
- 显卡型号是否和你描述的一致（比如真的是 RTX 4090 而不是其他型号）
- 显卡能不能稳定运行（跑压力测试不崩溃、不过热）
- 显存是否完整、没有问题

---

## 这个工具安全吗？

**安全。** 以下是明确承诺：

- **不会远程控制你的电脑** — 没有远程桌面、没有后门、没有网络连接
- **不会访问你的个人文件** — 不碰你的文档、照片、微信记录、密码
- **不会在后台运行** — 脚本跑完就结束，不安装任何常驻程序
- **不上传任何数据** — 所有结果只保存在你电脑的 `output/` 文件夹里
- **代码完全公开** — 所有 `.ps1` 文件都是普通文本，用记事本就能打开查看

如果你想自己查看代码，右键任意 `.ps1` 文件，选"用记事本打开"，里面写的都是中文注释，你可以看到每一步在做什么。

## 操作步骤（一步步来）

### 第一步：下载工具包

买家会给你发一个链接，或者你自己从 GitHub 下载：

```
https://github.com/JoeyYu23/gpu-check-protocol
```

点页面上的 **Code → Download ZIP**，下载后解压到桌面（或任意地方）。

### 第二步：下载测试软件

这些都是免费的专业硬件测试软件，各大硬件论坛都在用：

| 软件名 | 下载地址 | 怎么放 |
|--------|---------|--------|
| GPU-Z | https://www.techpowerup.com/gpuz/ | 下载 .exe，放进 `tools/` 文件夹 |
| HWiNFO64 | https://www.hwinfo.com/download/ | 下载 portable 版，放进 `tools/` |
| FurMark | https://geeks3d.com/furmark/ | 下载安装，安装目录选 `tools\FurMark\` |
| OCCT | https://www.ocbase.com/download | 下载安装，安装目录选 `tools\OCCT\` |

放好后，`tools/` 文件夹看起来应该像这样：

```
tools/
  GPU-Z.exe
  HWiNFO64.exe
  FurMark/
    FurMark.exe
  OCCT/
    OCCT.exe
```

### 第三步：运行验机

1. 找到解压出来的文件夹
2. **右键** `run_windows.bat`
3. 选择"**以管理员身份运行**"
4. 黑色命令行窗口会弹出，按照提示操作

> 为什么需要管理员权限？有些硬件信息（比如内存频率、主板型号）需要管理员权限才能完整读取。不以管理员运行也能跑，只是部分数据可能显示 N/A。

### 第四步：等待测试完成

程序会自动运行：
- **收集系统信息**（1分钟）：CPU、主板、内存、显卡型号、驱动版本
- **FurMark 压力测试**（5分钟）：显卡全力工作，测试稳定性
- **OCCT 显存测试**（10分钟）：需要你手动在 OCCT 里点一下开始按钮，脚本会提示你

期间电脑风扇会变响、温度会升高，这是正常的，不用担心。

### 第五步：发送结果

测试完成后，桌面会提示结果文件位置，类似：

```
output\gpu_check_20240315_143022.zip
```

把这个 **ZIP 文件**发给买家就行了。

## 各工具说明

- **GPU-Z**：行业标准显卡信息工具，读取显卡型号、显存大小、驱动版本，类似显卡的"身份证"
- **HWiNFO64**：专业硬件监控，每秒记录温度、风扇转速、功耗。测试完后买家可以看到温度曲线是否正常
- **FurMark**：俗称"甜甜圈"，是显卡圈最知名的烤机工具，让显卡全力工作，看会不会崩溃或过热
- **OCCT**：专门测试显存（VRAM），能发现显存颗粒坏点问题

## 常见疑虑解答

**"我担心这个程序是恶意软件"**

完全理解！建议你用 VirusTotal（https://www.virustotal.com）扫描 `run_windows.bat` 和 `scripts/` 文件夹里的 `.ps1` 文件。这些都是纯文本脚本，没有任何可执行二进制代码。

**"运行完要把这个软件卸载吗？"**

不需要卸载，直接删除整个文件夹即可（包括 `tools/` 里的工具）。没有安装任何东西到系统目录。

**"测试过程中显卡温度很高正常吗？"**

RTX 4090 在 FurMark 烤机时 GPU 温度 85-95°C 是正常范围，结温（Junction Temp）可以到 100°C 以下。如果风扇正常运转、测试没有中断，就没有问题。

**"买家能看到我电脑里的什么信息？"**

只有硬件信息：CPU 型号、主板型号、内存容量、显卡型号和驱动版本、磁盘容量和序列号。你的文件、账号、浏览器记录等完全不在采集范围内。

## 如果出现问题

遇到任何问题，把命令行窗口截图发给买家说明，或者查看 `output/` 目录里的 `session_transcript.log` 文件（用记事本打开），里面有完整的运行记录。
